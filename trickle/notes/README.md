# LinguaPlay - Language Learning App

## Описание проекта
Одностраничное приложение для изучения иностранных языков с элементами геймификации. Дизайн выполнен в голубых тонах с использованием персонажей Kirby и Cinnamoroll.

## Основные возможности
- **Полноэкранная прокрутка**: Каждая секция занимает весь экран с автоматической привязкой
- **Авторизация**: Система входа/регистрации с личным кабинетом
- **Личный кабинет**: Отображение прогресса, достижений и статистики обучения
- **Адаптивный дизайн**: Работает на всех устройствах
- **Анимации**: Плавающие персонажи и переходы
- **Ссылки на магазины**: App Store и Google Play

## Структура проекта
- `index.html` - Главная HTML страница с настройками Tailwind
- `app.js` - Основной компонент приложения
- `components/` - React компоненты:
  - `Header.js` - Шапка сайта с навигацией
  - `Hero.js` - Главная секция с персонажами
  - `Features.js` - Секция возможностей
  - `Download.js` - Секция загрузки приложения
  - `AuthModal.js` - Модальное окно авторизации и личного кабинета
